# RAPO Reporting Standard

## Purpose

All findings and scan results should be reported in a consistent, structured, and professional format.

## Required Finding Fields

Each finding should include:

- Finding
- Severity
- Description
- Evidence
- Recommendation
- References
- CVE
- CWE
- OWASP mapping

## Severity Levels

Use the following severity levels consistently:

- Informational: Low-risk observations with limited impact
- Low: Minor issues or weak signals
- Medium: Moderate risk requiring attention
- High: Significant security concern
- Critical: Severe issue with major impact or immediate risk

## Reporting Expectations

- Findings should be precise and evidence-based.
- Recommendations should be actionable.
- Reports should remain readable and professional.
- Where applicable, references should include standard sources such as CVEs, CWEs, and OWASP guidance.
