# RAPO Engineering Handbook

## Purpose

This directory contains the permanent engineering handbook for RAPO. These documents define how the project should be built, reviewed, and extended over time.

Every AI contributor — including GitHub Copilot, ChatGPT, Claude, Gemini, Ollama, Cursor, Windsurf, and other assistants — should read these documents before making changes to RAPO.
